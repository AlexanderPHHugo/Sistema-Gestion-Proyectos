# Sistema-Gestion-Proyectos

# Descripción del proyecto
Este proyecto es un sistema web desarrollado en PHP que permite la gestión de registros mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar), además de incluir autenticación de usuarios mediante login.
El sistema fue desarrollado en un entorno de servidor configurado manualmente con Apache y PHP. Este enfoque permitió comprender mejor la configuración real de un entorno de servidor web.

# Tecnologías utilizadas
  -PHP
  -MySQL
  -PDO (PHP Data Objects)
  -HTML
  -CSS

# Funcionalidades principales
  -Inicio de sesión de usuarios      
  -Manejo de sesiones en PHP
  -Crear registros (CRUD)
  -Listar registros en tabla
  -Editar registros existentes
  -Eliminar registros
  -Conexión a base de datos usando PDO

# Estructura del proyecto
  -index.php → Punto de entrada principal del sistema
  -/models → Contiene la lógica relacionada a la base de datos (consultas SQL, manejo de datos)
  -/controllers → Maneja la lógica de la aplicación y la comunicación entre modelos y vistas
  -/views → Contiene las interfaces del usuario (formularios, tablas, páginas)
  -/config → Archivo de configuración de la base de datos y conexión PDO
  -/assets → Recursos estáticos como CSS, imágenes y scripts JavaScript

# Aprendizajes
  -Configuración manual de entorno de servidor Apache + PHP
  -Modificación de archivos de configuración
  -Conexión a base de datos MySQL usando PDO
  -Manejo de sesiones en PHP
  -Implementación de operaciones CRUD
  -Estructuración básica de una aplicación web
